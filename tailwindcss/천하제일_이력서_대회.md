* github push 비중은 html 30% / css 30% / js 40%
* project는 총 4건
  - 1만시간의법칙(1일), 벤딩머신(2일), 렌딩페이지(3일), 감귤마켓(1주일)

* 별도 CSS 불허
* 심사는 디자인 뿐만 아니라 글귀도 판단합니다. 아주 단촐하고 진실되게 쓰는 것도 좋습니다. :)
